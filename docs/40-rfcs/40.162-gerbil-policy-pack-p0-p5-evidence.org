** Purpose
This note records the Gerbil policy-pack evidence used by the AgentGraph P0-P5 closure. It keeps complex policy composition in Scheme while preserving the native Scheme types to Rust types boundary.
** Native Receipt Boundary
The policy-pack projection chain is represented as POO receipts, not JSON:
- =module_evaluation_receipt=
- =policy_projection_receipt=
- =native_projection_payload=
- =budget_receipt=
- =catalog_resolution_receipt=
The direct projection helper exposes receipt family ids, nested receipt values, and owner fields as Scheme values that Rust can later consume through typed native projections.
