=marlin-gerbil-scheme=.

** Binding Shape

=gerbil-mysql= uses a three-layer binding shape:

- =db/_mysql.scm= owns the low-level Gambit FFI surface with =c-declare=,
  =c-define-type=, =c-lambda=, and helper macros such as =define-c-lambda=.
- =db/_mysql.ssi= owns the Gerbil interface boundary with =package:=,
  =export=, and =extern= declarations for the low-level functions.
- =db/mysql.ss= owns the public Gerbil API. It imports the low-level module and
  wraps pointer/error/async behavior behind Gerbil-level functions and structs.

The build script compiles the low-level FFI module first, builds the =.ssi=
interface, and then builds the public module:

#+begin_src scheme
(gsc: "db/_mysql" ...)
(ssi: "db/_mysql")
"db/mysql"
#+end_src

** Marlin Application

For Marlin, the same pattern maps to:

- a low-level native layer later, if we embed or link against Gerbil/Gambit
  directly;
- a Gerbil protocol binding layer now, where Gerbil owns constructors,
  accessors, and serializers for Marlin command protocol DTOs;
- a Gerbil request binding layer now, where Gerbil decodes
  =GerbilCompileRequest= JSON with =:std/text/json=;
- a Gerbil reader-backed parser layer now, where Gerbil compiles source text
  into protocol values;
- an adapter layer that reads requests, calls the parser, and emits one
  =GerbilCompileResponse=.

The current smoke implementation therefore keeps native VM bindings out of the
Rust crate while still moving bindings out of ad hoc JSON string construction.
The first binding module is:

#+begin_src text
crates/marlin-gerbil-scheme/fixtures/gerbil/marlin/protocol.ss
#+end_src

The first parser module is:

#+begin_src text
crates/marlin-gerbil-scheme/fixtures/gerbil/marlin/parser.ss
#+end_src

The first request module is:

#+begin_src text
crates/marlin-gerbil-scheme/fixtures/gerbil/marlin/request.ss
#+end_src

The first adapter module is:

#+begin_src text
crates/marlin-gerbil-scheme/fixtures/gerbil/marlin/adapter.ss
#+end_src

These modules use =package: marlin= and resolve through =GERBIL_LOADPATH= during
the smoke test.
